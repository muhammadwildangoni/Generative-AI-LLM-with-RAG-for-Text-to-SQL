# Generative-AI-LLM-with-RAG-for-Text-to-SQL
Sistem Text-to-SQL berbasis RAG menggunakan LLM (Gemini) untuk mengubah pertanyaan bahasa alami menjadi query SQL dan insight. Pipeline mencakup schema extraction, embedding (sentence-transformers), retrieval (Chroma), serta eksekusi query pada PostgreSQL menggunakan dataset Northwind.
